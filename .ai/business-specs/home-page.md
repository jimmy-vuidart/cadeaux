# Home Page

This page will simply display a page describing quickly the app, and allow with a button to create a new gift list. If the user is not logged in, clicking this button opens the account creation modal instead.

## Look & feel
- The home page welcomes the family with a joyful, gift‑themed ambiance.
- Buttons feel lively and inviting, with a subtle festive shine on hover and clear focus highlights.
- Sections are presented inside friendly, rounded cards. Some areas may use a celebratory style (gradient edges, confetti background) when appropriate to mark special moments like "Create a list".

## User Experience

### For Logged-In Users
- Displays a section titled "Mes listes" showing all lists created by the user.
- Each list is displayed with its title and the number of gifts it contains.
- A "Nouvelle liste" button allows users to create a new list.
- A "Partager" button is available for each list to share the list with others.
- Displays a section titled "Listes récemment visitées" showing lists that the user has visited but does not own.
- Each visited list is displayed with its title and the number of gifts it contains.
- A delete button is available for each visited list to remove it from the visited lists section.

### For Non-Logged-In Users
- Displays a "Créer ma liste de Noël" button that opens the account creation modal.
- Shows a brief description of the app's functionality.
- Displays a "Comment ça marche ?" section with a step-by-step guide:
  1. Create a new list for the event.
  2. Add all gift ideas.
  3. Share the unique link with family.
  4. Family members check off what they buy (user won't see anything).
