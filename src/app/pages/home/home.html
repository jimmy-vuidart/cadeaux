Cadeaux

Une petite application familiale pour gérer les listes de cadeaux et éviter les doublons.

Comment ça marche ?

  • Crée une nouvelle liste de cadeaux pour un événement (Noël, anniversaire, etc.).
  • Ajoute les idées de cadeaux dans la liste.
  • Partage la liste avec la famille pour que chacun puisse voir les idées.
  • Active le mode « remplissage » pour que les autres puissent marquer les cadeaux comme déjà achetés, sans gâcher la surprise.

Prêt à commencer ?

Tu pourras ensuite ajouter les cadeaux un par un et activer le mode « remplissage » pour la famille.
